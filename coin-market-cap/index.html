Coin Market Cap (CMC) Plugin

The Coin Market Cap (CMC) plugin is a powerful WordPress plugin for integrating live cryptocurrency market data into WordPress websites. It connects to a third-party API to generate an automatic crypto price listing that includes real-time value, coin market cap, tickers, and charts. You can select which data to show and sort it based on rank or volume. Furthermore, each cryptocurrency links to a single page created by default. You can customize its appearance using shortcodes or custom HTML.

Key Features:

  • Table of 2500+ Cryptocurrencies: Offers a comprehensive list of 2500 cryptocurrencies, each with detailed information, helping users make informed decisions based on the latest market trends.
  • Automatic Page Generation for Coin Details: Automatically generates detailed pages for CMC Currency Details and CMC Advanced Currency, making it easy to provide all the important information without needing to do it manually.
  • Conversion Calculator: Track any currency value by easily converting any amount from one cryptocurrency to another using the Calculator.
  • Historical Data: Provide a detailed record of past cryptocurrency prices, trading volumes, and market capitalization to gain information about particular cryptocurrencies over time.
  • Exchange List: Display real-time price data of exchanges, like trading pairs, 24-hour volume, and updates to help users make informed trading decisions.
  • Twitter Feed: Display real-time tweets from popular crypto accounts directly on your site, allowing users to stay updated with the latest news and market opinions.
  • Filter Pages (Favorites): Star individual pages, marking them as favorites. Easily filter and access these starred pages for quick reference.
  • Live Price Changes: Automatically updates cryptocurrency prices, providing real-time data for visitors to keep them informed about continuous market changes.
  • 24hr/7d/30d/1y Market Changes: Show percentage price changes for 24 hours, 7 days, 30 days, and 1 year as per your requirements, helping users to check short- and long-term market trends.
  • Categorize Coins: Assign custom categories to coins, and add multiple categories to a single coin to make navigation simpler.
  • Enable/Disable Coins: Easily enable or disable specific coins from the list by toggling them on or off as per your requirements.
  • Trading View Chart: Display real-time data by converting a single-page chart into a Trading View Chart, providing users with an overview of market trends.
  • SEO-Friendly (Yoast-Compatible): Fully supports Yoast SEO, ensuring each exchange page has a unique SEO title, meta description, and open graph (OG) tags for better search engine optimization.
  • Top Losers and Gainers: Highlight coins with the biggest price drops (top losers) and the ones with the largest price increases (top gainers) over a specific time period.
  • Single Page Designs: Display the data on a single page using two design options: Advanced Tab design and Regular Clean design as per your requirements.
  • API Type Selection: Choose between CoinGecko, CoinMarketCap, "CoinGecko + CoinMarketCap" or "CoinMarketCap + CoinGecko" based on your preferred data source.
  • Custom Coin Slug: Customize the coin slug to create SEO-friendly URLs and improve search rankings.
  • Affiliate Links: Add affiliate links to redirect users to specific pages for more detailed information.
  • Select Fiat Currency: Choose the currency in which to display coin values.
  • Coin Loading: Manage how many coins you want to show on each page and set the total number of coins to be loaded on each page.
  • Custom Description: Add custom description text to each coin listing to highlight key details or provide additional information.

  • Compatibility with Other Cryptocurrency Plugins: The Coin Market Cap plugin is compatible with all cryptocurrency plugins.
